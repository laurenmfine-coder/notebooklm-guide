Basic Prompts - Tables and Comparisons

Comparison Table

Best for: Distinguishing similar conditions, drugs, or concepts

Create a detailed comparison table for [list 3-5 related conditions/medications]. Include columns for: mechanism, presentation, key diagnostic findings, treatment, and important side effects/complications.

Drug Comparison

Best for: Pharmacology review, learning drug classes

Create a table comparing all drugs in the [drug class] category. Include columns for mechanism of action, indications, major side effects, and contraindications.

Disease Comparison

Best for: Differential diagnosis, distinguishing similar presentations

Compare [condition A] and [condition B] in a table format. Show differences in: pathophysiology, clinical presentation, diagnostic criteria, and management.

Lab Value Interpretation Table

Best for: Understanding diagnostic patterns

Create a table showing how [lab test] values change in different conditions. Include columns for the condition, expected value/pattern, and what it indicates.

Advanced Prompts - Algorithms and Frameworks

Algorithm Description

Best for: Clinical decision-making, management pathways

Generate a step-by-step algorithm for managing [condition/symptom]. Start from initial presentation and include decision points, diagnostic steps, and treatment pathways. Describe it clearly so I can draw it as a flowchart.

Diagnostic Algorithm

Best for: Workup approach, systematic evaluation

Create a diagnostic algorithm for evaluating [symptom/complaint]. Include initial assessment, key branch points based on findings, and when to pursue different diagnostic tests. Format so I can create a flowchart.

Visual Framework

Best for: Understanding complex relationships, organizing information

Describe a visual framework for understanding [topic]. Organize it hierarchically or in a logical flow that I can draw or diagram. Include all key components and how they relate.

Before/After Comparison

Best for: Pathophysiology, understanding disease progression

Describe the visual differences between normal [organ/system] and what happens in [condition]. Detail what changes occur that I should be able to visualize or diagram.

Concept Map Description

Best for: Connecting related concepts, seeing the big picture

Create a concept map showing how [topic] connects to related concepts. Describe the central idea and all branching concepts with their relationships.

Specialty-Specific Visual Prompts

Anatomical Relationships

Describe the spatial relationships of structures in [anatomical region]. Include what's superior/inferior, medial/lateral, and anterior/posterior to each structure so I can draw a diagram.

Pathway Diagram

Describe the [biochemical/physiological] pathway for [process]. Include all steps, enzymes/regulators, and where interventions (drugs, diseases) affect the pathway. Format for drawing as a diagram.

Timeline of Disease Progression

Create a timeline showing the progression of [condition] from initial pathology to clinical manifestations. Include what happens at each stage and when different symptoms/findings appear.
Pro Tip

After getting a table or algorithm description, redraw it by hand or in a diagramming tool. The act of recreating the visual helps cement the information in memory far better than just reading it.
