Clinical Rotations Strategy

MS3-MS4: Learning from patients and excelling on the wards

The Clinical Learning Shift

Clinical rotations flip the script. Instead of lectures and textbooks, you're learning from patients, residents, and attendings. The challenge: extracting maximum learning from every clinical encounter.

NotebookLM helps you process clinical experiences, prepare for patient encounters, and ace shelf exams.

Daily Clinical Workflow

Pre-Rounds (Morning)

  • Review your patients: Use NotebookLM notebooks for each patient's conditions
  • Anticipate questions: Generate Q&A for conditions you'll discuss
  • Prepare teaching points: Have 1-2 facts ready to contribute

Post-Rounds (Evening)

  • Log new cases: Brief summary of each interesting patient
  • Upload relevant resources: Guidelines, review articles for conditions seen
  • Generate audio for commute home: Review today's learning points

Weekend Review

  • Review all cases from the week
  • Generate comprehensive summaries
  • Identify knowledge gaps
  • Shelf exam practice questions

Case-Based Learning System

Creating Case Notebooks

For each rotation, build a case library:

  1. Create notebook titled "[Rotation] - Case Library"
  2. After each significant case, add a case summary
  3. Include: presentation, differential, workup, diagnosis, management, teaching points
  4. Generate audio overview weekly to review cases

Building Illness Scripts

Use NotebookLM to help build illness scripts for common presentations:

Ask NotebookLM: "Based on these resources, create an illness script for [condition] including: epidemiology, pathophysiology, typical presentation, key exam findings, diagnostic workup, treatment approach, and complications to monitor."

Shelf Exam Preparation

NotebookLM complements (doesn't replace) question banks:

Integration Strategy

  1. Do practice questions (UWorld, Amboss)
  2. Upload explanations for missed questions to NotebookLM
  3. Generate focused audio on your weak areas
  4. Listen during commute to reinforce gaps
  5. Use Q&A to test yourself on trouble spots

Rotation-Specific Tips

Internal Medicine

  • Build notebooks by chief complaint (chest pain, dyspnea, etc.)
  • Create differential diagnosis frameworks
  • Generate management algorithms

Surgery

  • Pre-op notebooks for common procedures
  • Post-op complication timelines
  • Anatomy review audio for OR preparation

Pediatrics

  • Developmental milestones study guides
  • Vaccination schedules
  • Pediatric-specific presentations of common diseases

OB/GYN

  • Pregnancy timeline and complications
  • Labor and delivery stages
  • Common GYN presentations

Psychiatry

  • DSM criteria comparisons
  • Medication class summaries
  • Therapy modality overviews
