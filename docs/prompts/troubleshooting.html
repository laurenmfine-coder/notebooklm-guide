Troubleshooting Prompts

Fix common problems and improve NotebookLM responses

When NotebookLM's responses aren't quite right, these prompts help you get better results.

Response is Too General

When NotebookLM gives vague or surface-level answers

Request Specificity

This is too general. Focus specifically on [narrow aspect]. Provide detailed, clinically relevant information with specific examples.

Ask for Depth

Go deeper into [specific aspect]. Explain the underlying mechanism, clinical implications, and advanced concepts.
Common Cause

Overly general responses often mean your sources are too broad. Try creating a new notebook with only the specific content you need, rather than uploading entire textbooks.

Response is Too Complex

When NotebookLM's explanation is over your head

Request Simplification

Explain this in simpler terms, using everyday analogies. Then explain it again using proper medical terminology.

Step-by-Step Breakdown

Break this down step by step, starting from the most basic concept and building up. Don't assume prior knowledge.

Need More Examples

When you understand the concept but need to see it applied

Request Clinical Examples

Provide specific clinical examples of when/how [concept] applies in patient care. Include real scenarios.

Multiple Scenarios

Show me 3 different scenarios where [concept] would be relevant. Include both typical and atypical presentations.

Response Seems Wrong or Incomplete

When something doesn't match what you've learned elsewhere

Request Sources

Which of my uploaded sources supports this information? Quote the relevant sections.

Check for Missing Information

What other important information about [topic] is in my sources that wasn't included in your previous response?

Compare Approaches

Are there different approaches or controversies regarding [topic] mentioned in my sources? What are the different perspectives?

Audio Overview Issues

When the generated audio isn't helpful

Audio Too General

Solution: Create a new notebook with only focused sources on your specific topic.

Before generating the audio, tell me: what are the main topics you would cover in an audio overview of these sources?

Audio Missing Key Topics

Solution: Use the chat to focus on specific topics before generating audio.

When you create an audio overview, make sure to emphasize [specific topic] and [specific topic]. These are the most important for my exam.

Format Issues

When the response format doesn't work for you

Request Different Format

Reorganize this information as a [table/numbered list/comparison/flowchart description]. Keep the same content but change the format.

Request Different Length

This is too [long/short]. Provide a version that is [more concise/more detailed]. Keep it under [X] words.

Common Problems and Solutions

Problem Likely Cause Solution
Responses are too basic Sources are introductory level Upload more advanced materials
Responses don't match my sources Too many conflicting sources Create focused notebooks with fewer sources
Audio is too long/unfocused Too many diverse sources Create topic-specific notebooks
Can't find specific information Information not in sources Upload additional relevant materials
Tables are incomplete Information gaps in sources Add sources that cover missing information
General Troubleshooting Strategy

When responses aren't helpful, work through this checklist: (1) Are your sources focused enough? (2) Is your prompt specific enough? (3) Are you asking for the right format? Most problems come from one of these three areas.
